Compute superposition

The browser still calls one API. The host still imports one federated module. The difference is that the module can hold multiple possible placements at once: origin, caller edge, or data region. A placement decision collapses that superposition into a running machine only when the target advertises the same MachineN shell as the prepared vmstate. The selected replica returns through the origin with an explicit host-mediated backhaul.
